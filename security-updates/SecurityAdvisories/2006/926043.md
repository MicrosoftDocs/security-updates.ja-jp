マイクロソフト セキュリティ アドバイザリ 926043
===============================================

Windows シェルの脆弱性により、リモートでコードが実行される
----------------------------------------------------------

公開日: 2006年9月29日 | 最終更新日: 2006年10月11日

要訳

|                                |                                                                                                                                                           |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| **お知らせ内容**               | セキュリティ更新プログラムの公開                                                                                                                          |
| **セキュリティ更新プログラム** | [セキュリティ情報 MS06-057](https://technet.microsoft.com/security/bulletin/ms06-057)                                                                      |
| **被害報告**                   | あり                                                                                                                                                      |
| **回避策**                     | あり                                                                                                                                                      |
| **対応方法**                   | [セキュリティ情報 MS06-057](https://technet.microsoft.com/security/bulletin/ms06-057) で提供しているセキュリティ更新プログラムをインストールしてください。 |

※ 上記の情報は、公開日または最終更新日の情報を基に作成しています。

マイクロソフトは脆弱性について公開された報告の調査を完了しました。 この問題に対処するために [MS06-057](https://technet.microsoft.com/security/bulletin/ms06-057) を公開しました。利用可能なセキュリティ更新プログラムのダウンロード先を含むこの問題に関する詳細情報は、[MS06-057](https://technet.microsoft.com/security/bulletin/ms06-057) をご覧ください。 この脆弱性は、Windows シェルのリモート コード実行の脆弱性 - [CVE-2006-3730](https://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2006-3730) として対処しています。
