Security Advisory

マイクロソフト セキュリティ アドバイザリ 961040
===============================================

SQL Server の脆弱性により、リモートでコードが実行される
-------------------------------------------------------

公開日: 2008年12月23日 | 最終更新日: 2009年2月11日

|                                |                                                                                                                                                           |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| **お知らせ内容**               | セキュリティ更新プログラムの公開                                                                                                                          |
| **セキュリティ更新プログラム** | [セキュリティ情報 MS09-004](https://technet.microsoft.com/security/bulletin/ms09-004)                                                                      |
| **被害報告**                   | なし                                                                                                                                                      |
| **回避策**                     | あり                                                                                                                                                      |
| **対応方法**                   | [セキュリティ情報 MS09-004](https://technet.microsoft.com/security/bulletin/ms09-004) で提供しているセキュリティ更新プログラムをインストールしてください。 |

※ 上記の情報は、公開日または最終更新日の情報を基に作成しています。

マイクロソフトは脆弱性について公開された報告の調査を完了しました。マイクロソフトはこの問題を解決するための [MS09-004](https://technet.microsoft.com/security/bulletin/ms09-004) をリリースしました。利用可能なセキュリティ更新プログラムのダウンロード先を含むこの問題に関する詳細情報は、[MS09-004](https://technet.microsoft.com/security/bulletin/ms09-004) をご覧ください。この解決された脆弱性は「SQL Server の sp\_replwritetovarbin の制限付きメモリの上書きの脆弱性 - [CVE-2008-5416](https://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2008-5416)」 です。
