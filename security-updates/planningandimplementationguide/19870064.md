Internet Security and Acceleration (ISA) Server
===============================================

ISA Server、多層防御ファイアウォール、Web キャッシュはセキュアで高速、かつ管理しやすいインターネット接続を提供するのに役立ちます。内部ファイアウォールのような ISA Server の計画、展開に関連したトピックについて最善策とガイダンスを紹介します。
